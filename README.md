# Explain This 🧠

An AI-powered website that explains any CS or math concept at different difficulty levels.

## Features

- 🟢 Simple — explained like you're 12, pure analogies
- 🟡 Student — clear explanations for university level
- 🔴 Technical — formal, precise, and in-depth
- 💡 Quiz mode — test yourself after learning
- 🕓 History — tracks your last 5 looked-up topics

## How to Use

1. Type any concept (e.g. "recursion", "eigenvalues", "Big-O")
2. Pick your difficulty level
3. Hit **Explain it** and read the explanation
4. Click **Quiz me** to test your understanding

## Built With

- HTML / CSS / JavaScript
- Groq API (LLaMA 3.3)
- Hosted on GitHub Pages
